WiFi Password Viewer

Antywirus może pokazywać że to wirus ponieważ sprawdza hasła zapisane w systemie.
The antivirus may show that it is a virus because it checks the passwords saved in the system.


Uruchom program przez cmd(terminal/wiersz poleceń) komendą cd downloads (jeśli masz plik w pobranych).
Run program from cmd(terminal) use command cd downloads (if you have file in downloads).

Kliknij ENTER i wyświetlą Ci się sieci oraz ich nazwy.
Click ENTER it will show you WiFi names & passwords.
